# Portafolio Digital Interactivo — Norman Valenzuela

Proyecto de la materia Expresion del Diseno Grafico I (UMECIT).

## Como abrir en VS Code
1. Abre esta carpeta en VS Code: `Archivo > Abrir carpeta...` (o `code .` en la terminal).
2. Instala la extension **Live Server** (Ritwick Dey).
3. Clic derecho sobre `index.html` > **Open with Live Server**.
   (Tambien puedes abrir `index.html` directo en el navegador.)

## Estructura
- `index.html`  -> el deck completo (10 slides, navegacion con flechas, demos y lightbox)
- `assets/`     -> las evidencias (paginas exportadas de tus PDFs)

Nota: las tipografias se cargan desde Google Fonts, asi que necesitas internet
para ver Schibsted Grotesk e Inter exactas. Todo lo demas funciona offline.
